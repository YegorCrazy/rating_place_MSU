Эта программа получает ваше место в списках абитуриентов МГУ из всех, кто подал заявление. Все настройки находятся в файле config.json, данные вводятся в кавычки.

Необходимые библиотеки: json, requests, lxml, bs4 (команда в командной строке для скачивания - pip3 install *имя_библиотеки*)

link - ссылка на сайт ЦПК с таблицей по вашему факультету
name - ваше имя (точно так, как написано в таблице)
num - ваше имя может встречаться на странице несколько раз, в этом параметре вы указываете, какое из вхождений следует рассматривать (нумеруя с 1)
